============
Installation
============

Call to actions can be installed via composer or downloaded from the `TYPO3 Extension repository <https://extensions.typo3.org/extension/call_to_actions>`_.

.. code-block:: bash

   composer req liquidlight/call-to-actions

Make sure the extension is enabled in the CMS -> Extensions -> Call to Actions
